# Clustering

We employed a hybrid clustering approach:

Step 1: complete-linkage hierarchical clustering to define initial centroids for fuzzy c-means (FCM) clustering
  - Matrices of z-scores (from the LIMMA results) were used for clustering
  - Only cluster features that were significantly different in at least one contrast for a particular sex
  - Clustering is done separately by sex
  - Hierarchical clustering is stable and able to extract meaningful clusters, unlike FCM which uses random features as the initial centroids

Step 2: FCM clustering (all features, regardless of significance)
  - Only 1 iteration of the FCM algorithm using the cluster centroids from hierarchical clustering. This is so that the cluster centroids are not driven toward zero. This flattening happens because most features have z-scores near 0.
